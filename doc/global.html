fillColor() → {function|Bucket}

Sets or retrieves the fill color generator function. Given a 0 <= level <= 100, the function should return a color (string) suitable for setting the fill's (an SVG path) 'fill' and 'stroke' attributes.
Source:
  • d3-bucket.js, line 263
Returns:
The fill color generator function if none was specified, or a reference to the current bucket.
Type
function | Bucket
Example
bucket.fillColor(function(level){ return "green"; })
var fillColor = bucket.fillColor()
fillColor(100)
