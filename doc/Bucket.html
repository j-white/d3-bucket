amplitude(amplitude) → {number|Bucket}

Sets or retrieves the amplitude of the fill's surface wave. The amplitude is made relative to the buckets height, where 100 denotes the full height of a bucket. The amplitude is also scaled based on the bucket current level, only a fraction of the amplitude is used when the level is low.
Parameters:
Name Type Argument Description

 The wave's amplitude, where 0 <= amplitude <= 100.
